특장점

4가지 SMART 핵심 기술

파라이엔티의 모든 전력조정기 기술이 집약된 차세대 플래그십 전력조정기

01 특허 보유

SMART MPD

최대전력분산제어 기능으로 다수의 전력조정기를 제로크로싱으로 제어 시 각 전력조정기 간의 전력분배로 메인 수전의 전력을 평활합니다. 역률 개선 및 피크 전력을 감소시킵니다.

02 특허 #10-1405243

SMART PLF

한 대의 전력조정기에 여러 개의 병렬 부하 연결 시, 부분적인 부하의 단선을 검출해 내는 기능입니다. 오토튜닝을 통해 부하 특성을 파악하여 정확한 감지를 실현합니다.

03 컬러 LCD · 18키 패드

SMART DISPLAY (PSD)

컬러 LCD와 18KEYPAD 적용으로 사용자 인터페이스를 개선했습니다. 자가진단, 테스트 모드, 용이한 파라미터 설정이 가능하며, 본체 부착 및 분리 장착이 모두 가능합니다.

04 복합 제어

SMART COMBINATION

하나의 몸체에 최대 단상 3대의 전력조정기를 탑재하여 비용과 공간을 절약합니다. 단상, 삼상 등 다양한 결선 방식에 대응하며 1P1L, 1P2L, 1P3L, 3P3L 등을 지원합니다.

RS-485 · Modbus-RTU 통신

최대 32대까지 멀티드롭 네트워크 구성 가능. 최대 115,200bps 통신 속도로 PLC, SCADA, HMI와 표준 연동됩니다. 32Bit CPU 탑재로 고속·고정밀 연산을 처리합니다.

SMART MPD (Max Power Distribution, 최대전력분산제어) 기능

1) 역률개선 효과

- 최대전력 분산제어는 다수의 전력조정기를 제로크로싱으로 제어 시 각 전력조정기 간의 전력분배로 메인 수전의 전력을 평활하는 기능입니다.

- 일반 제로크로싱 제어방식과 전력분산 제어 시 메인 수전의 파형은 다음의 그림과 같이 나타낼 수 있습니다.

SMART PLF (Partial Load Failure, 부분부하단선 감지) 기능

(특허 제10-1405243호 부분부하단선 검출기를 구비한 전력조정기 및 이의 부분부하단선 검출 방법)

- 한 대의 전력조정기에 여러 개의 병렬 부하 연결 시, 부분적인 부하의 단선을 검출해 내는 기능

- 스마트 전력 조정기는 오토튜닝을 통해 부하 특성 패턴을 획득하여 정확한 부분부하 단선 검출을 실현 하였습니다.

SMART DISPLAY (PSD, PARA SMART DISPLAY) 기능

- 컬러 LCD와 18KEYPAD 적용으로 사용자 인터페이스 개선

- 전력조정기의 자가진단 기능탑재, 테스트 모드실행 및 쉬운 파라미터 설정을 통해 고객의 편의성 제고

- 또한, PSD (PARA SMART DISPLAY)가 본체와 분리 판넬 외부에 장착이 가능하여 시인성 확보

PSD 본체 부착 시

PSD 분리 시

PSD (PARA SMART DISPLAY)

- 전압/전류/전력/저항(온도:선택시)을 표시하고 각 상의 파라메터를 순환함으로써 사용자의 편의성을 제고

- 입력 및 출력량을 그래프로 나타냄으로써 현시성을 증가 시킴

- 설정 유니트의 탈.부착이 가능하며, 유니트를 탈착하여 패널의 전면부에 설치 시, 전류계 및 전압계를 별도로 부착하지 않아도 되는 장점이 있음.

SMART COMBINATION

- 비용 및 공간 절약을 위해 하나의 몸체에 최대 단상 3대의 전력조정기를 탑재하여 각각의 입력신호에 따라 제어가 되며 모든 히터결선에 사용 할 수 있도록 설계.

- 단상 1대 사용 시

- 단상 2대 사용 시

- 두상을 제어하여 단상 히터1개 사용 시

- 삼상 2LEG 사용 시(제로크로싱만 사용가능)

- 안정성 향상

- 경제성 향상

- 단상 3대 사용 시

- 삼상 4선식 사용 시

- 삼상 3LEG 사용 시(스타 및 델타 결선)

- OPEN DELTA 결선 시

- 입력신호 각각 입수 또는 1개의 입력신호로 3대 제어 가능

SMART COMBINATION

- 각각의 알람을 사용자의 선택에 따라 3개의 릴레이 중 택일하여 신호 발생

- 알람 발생 시, 출력을 차단여부 고객이 선택

- 펜의 수명연장을 위해 고객이 지정하는 방열판의 온도에 따라 펜 구동

- 방열판 과열 방지를 위해 고객이 원하는 온도에서 경고 및 알람 발생 지정

- 알람의 종류

① 과전류  ② 휴즈 단선  ③ 히터 단선  ④ 방열판 과열  ⑤ SCR 소자소손  ⑥ U.L  ⑦ 메인전원 차단 알람  ⑧ FAN고장 알람  ⑨ 주파수Noise 알람  ⑩ PLF(부분부하단선 알람)

SMART 통신기능

- 통신방식 : RS-485 2 WIRE 반이중방식

- 통신속도 : 4800, 9600, 19200, 38400, 57600, 최대 115200bps

- 최대 접속가능 대수 : 32대

- 프로토콜 : Modbus-RTU Protocol

- 통신을 통한 READ & WRITE 기능 실행

- USER INTERFACE 제공

SMART 테스트 기능

강력한 32Bit CPU(12Bit ADC)를 탑재하여 연산 및 반응속도를 높이고 최상의 제어 성능유지

기기에 전원 및 부하가 연결된 상태에서 현재 설정된 각종 파라미터 값과 알람 및 신호선 결선상태를 확인할 수 있는 기능 탑재

실부하로 테스트하기 어려운 경우, 작은 용량의 부하를 결선하여 사용자가 임의로 출력을 내보내면서 기기의 출력상태를 점검할 수 있는 테스트 기능 탑재
